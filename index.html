Download

Version 1.0

  • Linux

Installation

Windows

Entpacke das zip-Archiv in einen Ordner. Danach starte einfach tiere.exe.

Linux

Entpacke das tar-Archiv. Dafür kannst du den Befehl tar -xvf tiere-1.0.tar.gz verwenden, oder ein Entpackprogram verwenden. Danach kannst du einfach das Program tiere starten.

Tiere hinzufügen

Um Tiere hinzuzufügen muss nur die Datei tiere.json angepasst werden. Dabei steht jedes Tier zwischen zwei geschwungenen Klammern { }, die mit Kommas getrennt werden. Ein Eintrag für ein Tier sieht immer wie folgt aus:

{
"name" : "Tiername",
"image" : "Link zum Bild des Tiers (Local oder Internetlink)",
"video" : "Link zum Yotube-Video des Tieres",
"audio" : "Link zur Audio-Datei des Tieres",
"description" : "Beschreibungstext des Tieres"
},

Ein Beispiel für einen neuen Eintrag könnte z.B. so aussehen:

{
"name" : "Der Bär",
"image" : "http://upload.wikimedia.org/wikipedia/commons/c/c6/Kodiak_B%C3%A4r.jpg",
"video" : "https://www.youtube.com/embed/95I81SXjZLA?autoplay=1",
"audio" : "./audio/baer.mp3",
"description" : "Der Bär ist ein sehr großes Tier."
},
